# 멍로그 (MongLog)

유기견 보호소 서비스 웹 애플리케이션

## 주요 기능

- 🏠 **보호소 찾기**: 지역별 유기견 보호소 검색 및 필터링
- 🐕 **분실/보호 게시판**: 실종 및 보호 동물 정보 공유
- 💝 **분양하기**: 유기견 입양 정보 제공
- 🎨 **3D 인터랙티브 UI**: Three.js를 활용한 동적인 홈 화면

## 기술 스택

- React 18
- Three.js (r128)
- Tailwind CSS
- Lucide Icons

## 실행 방법

`index.html` 파일을 브라우저로 열면 바로 실행됩니다.

또는 로컬 서버 실행:
```bash
# Python 3 사용
python -m http.server 8000

# Node.js 사용
npx serve
```
